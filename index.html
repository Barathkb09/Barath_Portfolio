Realtime Chatting Application

This is a realtime chat application built using the MERN stack, which includes MongoDB, Express.js, React.js, and Node.js. The application allows users to register, log in, and communicate instantly with others through one-on-one chat. Real-time messaging is achieved using Socket.io, enabling smooth and instantaneous communication between users. The backend handles user authentication securely using JSON Web Tokens (JWT) and bcrypt for password hashing, while all user data and messages are stored persistently in MongoDB. The frontend, built with React.js, offers a responsive and intuitive user interface suitable for both desktop and mobile devices.

Quiz Application

This is a dynamic Quiz Application built using PHP and MySQL, designed to support both user and admin functionalities. The application features a secure login system for users and administrators, enabling controlled access based on roles. Regular users can log in, participate in quizzes, and receive their scores instantly upon submission, providing immediate feedback and enhancing the learning experience. On the other hand, the admin panel allows administrators to manage all aspects of the quiz system, including adding, updating, or deleting quiz questions, managing quiz categories, and monitoring user details and their performance records. All these records can be stored in database.

Responsive E-commerce Store

This is a full-featured E-commerce Store built using the MERN stack (MongoDB, Express.js, React.js, and Node.js), integrated with Razorpay for secure payment processing in test mode. The application offers a seamless shopping experience, allowing users to browse products, add them to a cart, and complete purchases through a streamlined checkout process. The store features user authentication and authorization using JWT, enabling secure account management and protected admin routes. Users can register, log in, and view their order history, while admins have access to a dedicated dashboard to manage products, categories, and orders. And admin can monitor the User and product analytics.
